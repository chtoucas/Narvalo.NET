Struct Fallible<T>

Represents the outcome of a computation which might have thrown an exception. An instance of the Fallible<T> class contains either a T value or the exception state at the point it was thrown.

public struct Fallible<T> : IEquatable<Fallible<T>>, IResult<T, ExceptionDispatchInfo>, IContainer<T>, ISecondaryContainer<ExceptionDispatchInfo>, Iterable<T>
Type Parameters
Name Description
T

The underlying type of the value.
Remarks

We do not catch exceptions throw by any supplied delegate. A good pratice is that a delegate should not normally throw.

This class is not meant to replace the standard exception mechanism.
