Either<TLeft, TRight>

Represents the sum of two types. An instance of the Either<TLeft, TRight> class contains either a TLeft value or a TRight value but not both.

This class is a "monad" for the left type parameter, nevertheless using Swap() or SwapUnchecked() you can easily turn it into a "monad" for the right type parameter.
