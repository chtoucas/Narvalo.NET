Namespace Narvalo.Web.Optimization

The Narvalo.Web.Optimization namespace contains types that enable you to optimize web content for HTTP delivery.

Work related to HTML optimizations would have not been possible without other open source contributors. I owe a lot to this blog to understand the ASP.NET WebForms side of things and Meleze.Web served as a great inspiration while trying to understand how something similar could be done with Razor.
