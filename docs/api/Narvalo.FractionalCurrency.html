Struct FractionalCurrency

Represents a subunit of a currency such as the EUR Cent or the GBP Penny.

Remarks

A subunit is a fraction of the main unit. The minor currency unit (when it exists) is such an example. It is possible for a currency to have more than one subunit.

Contrary to a Currency, we do not restrict ourselves to subunits for which Factor is a power of 10.
