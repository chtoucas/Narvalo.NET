Enum CurrencyTypes

Defines the types of currencies that can be used to create a currency using the Of(String, CurrencyTypes) and TryCreate(String, CurrencyTypes) methods.

Namespace:Narvalo
Assembly:Narvalo.Money.dll
Syntax
[Flags]
public enum CurrencyTypes
Remarks

This enumeration has a System.FlagsAttribute attribute that allows a bitwise combination of its member values.

Fields

Name Description
Active

All circulating ISO currencies.
Any

All supported currencies.

Any is a composite field that includes the Active, Withdrawn and UserDefined fields.
Circulating

All circulating currencies whether they are ISO or not.

Circulating is a composite field that includes the Active and UserDefined fields.
ISO

All ISO currencies.

ISO is a composite field that includes the Active and Withdrawn fields.
UserDefined

User-defined currencies (not part of ISO 4217).

For instance, this is the case for crypto-currencies like the Bitcoin (XBT), unofficial codes like the Jersey Pound (JEP), currencies issued by an unrecognized state, and so on.
Withdrawn

ISO currencies that have been superseded or simply deleted.
